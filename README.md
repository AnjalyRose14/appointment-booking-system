# Appointment Booking System

A full-stack appointment booking system built using Java and Spring Boot. 
The application provides secure user authentication, role-based access control, 
and RESTful APIs for managing appointments and time slots.

## Features
- User registration and login
- JWT-based authentication and authorization
- Role-based access (User / Admin)
- Appointment booking and slot management
- RESTful API design
- Secure backend architecture

## Tech Stack
- Java
- Spring Boot
- Spring Security (JWT)
- REST APIs
- MySQL
- Git & GitHub
- Postman
